

# TRIA Health — Professional HealthTech Landing Page

A cinematic, tech-savvy landing page combining the restrained elegance of Neuralink/Wing/Skydio with a healthtech identity. Uses a subtle teal/cyan accent color alongside the grayscale palette for a medical-tech feel without breaking the professional tone.

## Design System
- **Primary palette**: Deep charcoal (#0F0F0F), off-white (#F5F5F5), medium gray (#444)
- **Accent color**: Muted healthtech teal (#0A9396) — used sparingly on CTAs, key stats, and the dual-layer diagram connectors
- **Typography**: Inter — heavyweight tight-tracked headlines, clean body text
- **All media**: Placeholder images/videos with cinematic overlays, autoplay muted loops
